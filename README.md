# Deb8 - Backend

## Requirements
- Python version 3.10.12
- Docker newest version + compose plugin

## Instalation

```shell
python3 -m venv .venv

# windows
.venv/Scripts/activate
# linux
source .venv/bin/activate

pip3 install -r requirements.txt
```

## Running project
```shell
docker compose up -d --build
```
